Malai Events

Ideathon

Ideathon 2022 is a hybrid hackathon conducted by the Malai club at Anurag University. The students presented their ideas that could be incorporated into realworld problems related to AIML in this ideathon. Participants worked in teams and used innovative ideation practices to brainstorm on possible solutions.


Car Automation

Drive into automation is a hands-on learning experience that focuses on automating a car model using AI technologies such as OpenCV and TensorFlow. We conducted a training session for students on how to implement car automation using various Python packages.


OpenCV

OpenCV workshop is a hands-on learning experience that focuses on a common infrastructure for computer vision applications using AI technologies. We conducted a training session for students on how to implement criminal detection using various Python packages.


ANN-Microsoft

Artificial neural network is a hands-on learning experience that focuses on how computers make intelligent decisions with limited human assistance. This is because they can learn and model the relationships between input and output data that are non-linear and complex.

AI MED QUEST
